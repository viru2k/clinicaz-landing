¿Por que deberias usar este software?

La arquitectura de este software esta alojada en la nube, con esta tecnólogia podes consultar tu agenda y ver tus historias clínicas donde quieras y desde cualquier dispositivo

Sí, dependiendo del módulo que elijas puedes administrar distintas clinicas y tener tus agendas para cada una de ellas. Incluso puedes ampliar la cantidad de clínicas

Podes elegir que usuario es un "Médico" y asignar roles a cada usuario (Ejemplo : Una secretaria podra dar turnos, pero no podra modificar historias clinicas), si contas con personal administrativo podes asignarle roles de facturación , auditoria y liquidación


Cada módulo tiene una cantidad de usuarios que puede ser ampliada. Puedes combinar los usuarios y clínicas para tener tu agenda organizada


El módulo "CLINICAS" te permite realizar cobros, cargar ordenes y obtener un resúmen de tus consultas. Asi siempre sabras tu estado financiero


Puedes realizar operaciones de cobro de obras sociales y agruparlas para saber cuantas ordenes presentaron. clinicAZ posee un módulo muy potente para gestionar todas las obras sociales con sus respectivas modalidades
